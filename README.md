--- ゲ-ム説明 ---
  
  外側に逃げようとする赤い球を黒い球で囲むゲ-ムです
  
  赤玉が端に行ったらゲ-ムオ-バ-


-----------------------------------------------------------------------------------------------


  赤玉の最短距離の計算方法の流れ

 1 円にIDを振り当てる
 
 2 隣接行列を作る
 
 3 幅優先探索により探索する
 
 4 3での結果をもとにそれぞれ道を一つに繋げる
 
 5 4の中で一番距離が短いものを選ぶ(複数ある場合はランダムで選ぶ)
 
 
 ----------------------------------------------------------------------------------------------
